# Como usar este Brain

## Abrir no Obsidian

1. **File → Open vault → Open folder as vault** → selecione `D:/Pessoal/Sonar/message_sending/obsidian_sonnar/`.
2. Comece por [[brain]].

## Convenções

- Wikilinks `[[...]]` para navegar.
