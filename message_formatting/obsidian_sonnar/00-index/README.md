# Como usar este Brain

## Abrir no Obsidian

1. **File → Open vault → Open folder as vault** → selecione `D:/Pessoal/Sonar/message_formatting/obsidian_sonnar/`.
2. Aguarde indexação.
3. Comece por [[brain]].

## Convenções

- Wikilinks `[[...]]` para navegar.
- Frontmatter com `tags` para filtrar.
- Cite caminho relativo do código (`core/src/...`, `whatsapp/src/...`) ao referenciar.

## Manutenção

- Novo template/canal? Adicione nota em `02-domains/` e linke em `02-domains/index.md`.
- Decisão técnica importante? `12-decisions/ADR-NNN-slug.md`.
- Bug recorrente? `13-issues/`.

## Padronização

Este vault segue a mesma estrutura numerada (00..15) dos outros 3 vaults do monorepo:
data_collection, landing-page, message_sending.
